
# Google Keep Clone

I built this project as a fully functional clone of Google Keep. I used React (with Vite) to build the frontend, focusing heavily on robust state management and precise CSS styling.

## Features

I implemented the following key features:
- **Interactive Note Management:** I added the ability to create, delete, and color-code notes so that users can organize their thoughts easily.

- ** Using Cursor AI**
- **Real-Time Reminders :** 
- **Drag-and-Drop Reordering:**
- **Responsive Navigation:** I built a fully responsive sidebar navigation system that gracefully collapses so that it works mobile devices aka smaller desipalys.
- **State Management & Persistence:** I utilized React's state management to ensure that note operations are smooth and that interactive UI components update seamlessly.
- **Styling:** I used detailed CSS (along with media queries) to achieve precise icon positioning and ensure cross-device compatibility.


## Project Structure

I organized my components primarily in the `src/containers/` folder, which includes modular components like `Main.jsx`, `Aside.jsx`, `SearchBar.jsx`, and `Navigation.jsx`. I matched these components with their respective CSS files located in `src/styles/` to keep everything as clean as possible. 



A lot of effort was also was into making the UI feel extremely close to the actual Google Keep app, and I'm very proud of how the interactive note management turned out.
